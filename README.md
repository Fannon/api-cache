> A simple Node.js ASK Query Cacher (for Semantic MediaWiki)


## Install

```sh
$ npm install --save smw-cacher
```


## Usage

```js
var smwCacher = require('smw-cacher');

smwCacher('Rainbow');
```

```sh
$ npm install --global smw-cacher
$ smw-cacher --help
```
